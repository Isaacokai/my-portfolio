Sample Bug Report

Example of the structure you’ll receive.

Title
Checkout button remains disabled after selecting shipping method
Environment
Web • Chrome 122 • Windows 11 • Staging
Steps to reproduce
  1. Add any item to cart
  2. Go to Checkout
  3. Select a shipping method (e.g., “Standard”)
  4. Try to click “Place Order”
Expected
“Place Order” becomes enabled once a shipping method is selected.
Actual
“Place Order” stays disabled; user cannot complete checkout.
